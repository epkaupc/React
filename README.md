# Mouse Teleport Button

A fun little project where a button dodges the user's cursor by teleporting to random spots on the screen causing a yes/sim loop
## Run in website
  https://whyme44.netlify.app
## What it does

- Tracks mouse position in real time
- Detects when the cursor touches the button
- Teleports the button to a random location
- Changes the button on hover for a playful effect

## Built with

- HTML
- CSS
- JavaScript
- Reactjs
